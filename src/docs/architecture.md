# Architecture view interpretation

The architecture view is intentionally not just a curved-line grid.

## Design rule

- **Columns** anchor OEM groups.
- **Rows** anchor stack layers.
- **Node stacks** group entities inside lanes.
- **Edges** express cross-layer and cross-entity relationships.

## Available controls

- **Adjacent Architecture Links** narrows the display to near-layer relationships for cleaner stack reading.
- **Supplier Emphasis** keeps the architecture focused on external ecosystem dependencies.
- **Tight Architecture Lanes** reduces lane density for faster executive scanning.

## Evidence-aware styling

Verified relationships appear stronger and cleaner. Inferred or weaker relationships appear lighter or dashed so the visual system does not imply false certainty.
